# 关于Springboot2-demo-generator 
    这是一个关于web页面在线生成dao,mapper文件等的实验型项目.核心原理是利用mybatis-generator-core进行生成.
    
## 支持的数据库版本,以及对应驱动

| 支持的数据库 | 数据库版本         | 驱动名称             | 驱动版本 |
|:---|:---|:---|:---|
| Mysql        | 5.5, 5.6, 5.7, 8.0 | mysql-connector-java | 8.0.13   |
| Oracle | 常见版本 | ojdbc6 | 11.2.0.3 |
